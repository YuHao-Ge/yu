# Way 1:下载GitHub下所有的项目文件
点击其主页下的Clone or download下的Download Zip ，可以直接下载到本地电脑；
或者复制其链接，命令行 git clone [URL] ，下载到当前路径

# Way 2:下载指定文件夹
step 1：From GitURL to SVNURL
https://github.com/michaelliao/learn-python3/tree/master/samples/multitask
--将”/branches/branchname/”替换成”/trunk/”-->
https://github.com/michaelliao/learn-python3/trunk/samples/multitask

step 2：命令行 svn checkout [SVNURL]

#Way 3：下载指定文件
step 1：创建空目录

step 2：cd [空目录]
	svn up [文件名]
